# ZDRIVE 

> Drive, but cooler.

8ms cached response · ∞ bandwidth · 100K free requests/day

## Features Detail

### Browse & Search
- Intuitive folder navigation with breadcrumb
- Instant local search + deep global search
- Smart filters (folder, file, video, image, audio, PDF, docs)
- Sort by name, date, or size

### Preview & Play
- **Images:** Zoom with scroll wheel, pinch to zoom, pan & drag
- **Videos:** Quality selector (240p-4K), speed control (0.25x-2x), PiP mode
- **Audio:** Speed control, loop mode
- **PDF:** Inline viewer with navigation
- **Markdown:** Live preview with syntax highlighting
- **Code:** Syntax highlighting for 20+ languages

### Download & Share
- Direct file download
- Folder download as ZIP (max 100 files / 200MB)
- One-click copy share links

### Experience
- Auto dark/light mode with manual toggle
- Fully responsive (desktop, tablet, mobile)
- Keyboard shortcuts for power users
- Edge cached for sub-second response
